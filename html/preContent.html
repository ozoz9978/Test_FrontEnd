pre 요소

일반적으로는 탭, 스페이스, 엔터가 아무리 요소 내에 많아도 하나로 처리한다.
프로그램 등의 코드를 화면에 나타내야 할 때에는 pre 태그가 효과적이다.

pre가 없는 코드(아래의 코드는 추천하지 않는다.)

pulbic class Test{
    public static void main(String[] args) {
        System.out.println("Hello, world");
    }
}

pre가 있는 코드

        pulbic class Test{
            public static void main(String[] args) {
                System.out.println("Hello, world");
            }
        }
