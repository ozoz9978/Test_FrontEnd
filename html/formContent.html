폼요소

사용자의 입력을 받아 그 내용을 서버로 전송하기 위해 사용하는 태그
폼요소는 대부분 인라인 요소
회원가입, 로그인, 비밀번호 변경, 게시글 입력, 댓글 입력 등에서 사용

input, textarea, select, label 등이 있다.
name 속성 : 서버측 변수명의 역할

form 태그의 속성 중
1) method="get" : 생략 가능, 주소 표시줄에 노출됨, 다른 화면을 달라는 요청. 많은 양의 데이터 전송시에는 get 사용하지 않는다.
2) method="post" : 생략 불가, 주소 표시줄에 노출되지 않음, 처리 요청 많은 양의 데이터 전송, 파일 첨부시 post 사용

여러 개에서 선택 : radio, checkbox, select

여러 항목 선택(지역, 취미)
