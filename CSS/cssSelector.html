CSS 선택자

요소에 서식을 적용하기 위해서는 요소를 정확히 선택하는 것이 매우 중요!
선택자(Selector)는 요소를 다양하게 선택하도록 제공하는 방법

  • 단순선택자
    • 전체 선택자 : * (특이값 0)
    • 요소 선택자 : 태그명 그 자체 (특이값 1)
    • 그룹 선택자 : , (콤마)
    • 클래스 선택자 : class 속성을 주고 (,클래스명) 여러 요소를 선택 (특이값 10)
    • 아이디 선택자 : id 속성을 주고 (#아이디명) 하나의 요소를 선택 (특이값 100)
    • (예) 단순선택자의 사용
  • 가상(Pseudo) 클래스 선택자
    • (참고) w3schools 가상 클래스
    • 가상 클래스는 특정요소의 상태에 따라 선택하는 방법
    • "요소명 : 상태" - 주로 a 태그의 상태를 나눠서 가상클래스를 적용하지만 다른 태그도 가능
    • (예) 가상 클래스 선택자의 사용
  • 가상(Pseudo) 요소 선택자
    • w3schools 가상요소선택자의 사용
    • 가상요소란? 태그의 지정된 부분에 스타일을 적용할 때 사용
    • 콜론 2개 (::)를 붙여서 요소의 특정부분에 스타일을 적용
    • (예) 가상 요소 선택자의 사용
    • 연습 테이블에 가상클래스 선택자
  • 속성선택자
    • html 요소 중에서 특정 속성(attribute)를 가진 요소를 선택
    • (참고) w3schools 속성선택자
    • [속성] : 요소가 지정된 속성이 있을때
      [속성 = 값] : 요소의 속성이 지정된 값을 가지고 있을 때
      [속성 ~= 값] : 특정 단어를 가지고 있는 경우
      [속성 ^= 값] : 특정값으로 시작
      [속성 $= 값] : 특정요소의 끝나는 글자
      [속성 |= 값] : 지정된 값(단어)를 포함하거나 단어 뒤 -(하이픈)으로 되어있을때
      [속성 *= 값] : 지정된 값을 포함(단어가 아니어도 됨) 하고 있을때
    • (예) 속성 선택자의 사용
  • 종속선택자
    • 둘 이상의 선택자를 조합해서 사용(족보)
    • 공백(후손선택자):하위 항목 전부
    • >(자식선택자): 인접하게 바로 포함된 항목을 선택
    • + (형제선택자): 상위클래스가 같으면서 연달아 있는 항목을 선택
    • ~(일반형제선택자): 같은 상위클래스를 가진 항목을 선택
    • (예) 종속 선택자의 사용
  • 기타선택자
    • not(선택자) : 괄호안에 지정한 선택자가 아닌 요소들을 선택
    • nth-child(수식) : 수식에 맞는 요소를 선택
    • (예) 기타 선택자의 사용
