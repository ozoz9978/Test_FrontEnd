폰트 관련 스타일

폰트

  • font-family: 글꼴지정, 대중글꼴을 사용할 것
  • font-size : 글꼴 크기 지정
  • font-weight: 굵기 (100~900), 400(기본), bold, bolder
  • font-varient: 영문자 기준. 소문자 크기로 대문자 표현

텍스트

  • color:글자색상
  • text-decoration : 밑줄 등을 넣거나 뺄때
  • text-transform : 대소문자 변경
  • text-shadow : 그림자
  • white-space : 공백처리
  • letter-spacing / word-spacing : 공백처리
  • direction : 방향
  • text-align : 수평정렬
  • text-justify : 정렬 시 남는 공백 처리
  • line-height : 줄간격
  • text-overflow: overflow 속성과 함께 사용하고 영역내에 글자가 흘러넘칠경우 표현방법
