position을 이용한 배치

static : 기본값, absolute : 하나의 레이어를 생성, 위치지정은 top, left, right, bottom
  부모의 position위치를 기준으로 이동
  만약 부모가 position 이 없으면 body가 기준이 된다.
relative : 하나의 레이어를 생성, 위치지정은 top, left, right, bottom
  해당 해당요소 자체 기준으로 이동
fixed : 하나의 레이어를 생성, 위치지정은 top, left, right, bottom
  화면을 기준으로 이동
